# fullstack_developer_capstone

Repository name: xrwvm-fullstack_developer_capstone

Project name: fullstack_developer_capstone

This project is a Django and React based full stack developer capstone project.

## Project Details

The application allows users to:
- View all car dealers
- Filter dealers by state
- View dealer details and reviews
- Register and login
- Submit dealer reviews
- Access About Us and Contact Us pages

## Technologies Used

- Django
- Django REST Framework
- React
- HTML
- CSS
- GitHub Actions
